# UI reference assets

이 폴더는 Stitch에서 받은 화면 참고 자료를 보관한다.

현재 기준 자료는 `003.스티치`다. `001.스티치`, `002.스티치`는 과거 참고 묶음 이름으로 남아 있을 수 있지만, 현재 커밋 기준에는 실제 파일이 있는 `003.스티치` 자료만 포함한다.

역할:
- 앱 기능 구현 기준이 아니라 화면 밀도, 배치 리듬, 버튼 우선순위, 색감, 모바일 흐름을 판단하는 visual reference다.
- 산식, 점수, Records 저장 구조, InBody gate, backup 정책을 바꾸는 근거로 쓰지 않는다.
- 실제 구현 판단은 `docs/README.md`, `docs/개발정책.txt`, `docs/캔버스_updated.txt`, `docs/v8_운동프로필_산식_정밀설계.txt`를 우선한다.

주의:
- `code.html`은 참고용 산출물이며 앱 코드로 복붙하지 않는다.
- `screen.png`는 렌더 비교용 참고 이미지다.
- 빈 폴더는 git에 저장되지 않는다.
