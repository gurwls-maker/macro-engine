# 탄단지 다이어리 v6.9 Mobile Redesign Brief

탄단지 다이어리는 매일 식사, 탄수화물, 단백질, 지방, 체중, InBody 기록을 입력하고 목표 대비 결과와 코치 문구를 확인하는 모바일 중심 건강관리 앱입니다.

## 유지해야 할 구조

앱은 반드시 5개 주요 탭 구조를 유지합니다.

1. 오늘 계산
2. 기록
3. 최근 흐름
4. InBody
5. 기본 설정

새 탭을 만들거나 기존 탭을 합치지 말고, 각 탭 안의 화면 구성과 시각 디자인을 개선해주세요.

## 리디자인 목표

기능, 입력 항목, 계산 결과, 코치 문구, 모달 흐름은 유지하고 겉모습과 모바일 레이아웃 완성도를 높여주세요. 첨부 이미지는 같은 앱의 여러 상태와 트리거 화면입니다. 한 화면만 예쁘게 바꾸지 말고 전체 앱의 공통 컴포넌트 규칙을 잡아주세요.

## 화면별 중요 요소

- 오늘 계산: 오늘 입력, 목표 대비 결과, 코치 문구, 빠른 수정, 기록 시작 바텀시트
- 기록: 날짜별 기록, 상세 보기, 기록 수정, 식사 추가/수정 모달
- 최근 흐름: 1/3/7/14/28일 범위별 분석, 저데이터 상태, 코치 해석
- InBody: 신규 입력, 기존 기록 수정, 추세, 기록 연결
- 기본 설정: 목표, 프로필, 생활 패턴, 운동 설정, 전문가 설정, 백업/복원

## 시각 방향

차분하고 신뢰감 있는 green / ivory 계열의 모바일 헬스 도구로 리디자인해주세요. 짙은 초록, 부드러운 잎색, 아이보리 배경, 얇은 라인, 절제된 그림자를 사용합니다. kcal, 목표 달성, 진행률은 amber 계열을 보조로 사용하고, 경고는 coral/red를 제한적으로 사용합니다.

## 피해야 할 것

보라/파랑 그라데이션 위주의 SaaS 느낌, 장식용 배경, 과한 카드 중첩, 랜딩페이지 같은 히어로 구성, 새 기능 추가, 기존 입력 항목 삭제를 피해주세요.

## 모바일 기준

기본 모바일 기준은 390px 폭입니다. 오른쪽 잘림, 가로 스크롤, 버튼 텍스트 넘침이 없어야 합니다. 긴 입력 화면과 모달에서도 주요 버튼과 정보 구조가 안정적으로 보여야 합니다.
