## 8. Regression implication

Existing `runMacroRangeContinuousScoringTests` already covers the most important relation guards:
- TDEE overload raw score keeps worsening after final clamp
- fat excess has no fixed plateau and extreme fat can curve-collapse
- carb exercise context modifies curve without exercise bonus
- unknown carb context stays between rest and training
- alcohol kcal and physiology risk are separate monotonic axes
- training context does not discount alcohol physiology
- adaptive target exchange uses continuous recent excess pressure

Therefore, this branch does not need to add a new test suite before closing the decision.

If a future score tuning implementation changes any coefficient, that implementation must add before/after regression cases for the exact changed curve.
