## 5. production score formula assertion

formula:

```text
rawScore = 100 + sum(kcal, protein, fat, carbExchange)
productionScore = clamp(rawScore, 0, 100)
```

normal value domain:

```text
integer 0..100
null
```

rounding:

```text
identity / no-op integer
fractional input은 round/floor/ceil로 복구하지 않고 invalid로 reject
```

upper clamp:

```text
positive contribution을 조용히 100으로 흡수하지 않는다.
positive contribution은 invalid payload로 reject한다.
```

lower clamp:

```text
approved numeric path에서 rawScore < 0이면 0으로 clamp한다.
현재 v0 exact table의 정상 조합은 대부분 0 미만으로 내려가지 않지만,
defensive boundary는 test fixture로 유지한다.
```
