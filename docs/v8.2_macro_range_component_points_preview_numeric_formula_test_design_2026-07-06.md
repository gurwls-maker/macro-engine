## 4. exact value assertions

future suite는 severity label만으로 전역 정렬하지 않는다.

검증 기준:

```text
component + status + status-local evidence별 exact expected value를 검증한다.
```

이유:

```text
anchor material kcal severe -16은 range_evidence strong fat -12보다 크다.
따라서 strong / material / observation label만으로 전역 크기 순서를 assert하면 안 된다.
lane principle은 설명용 guard이고, actual expected value는 component/status table이 source of truth다.
```
