## 6. adaptive target overfitting 재판정

사용자 우려:

```text
먹은 만큼 목표가 계속 올라가면 앱이 과식을 합리화할 수 있다.
운동일 고탄수 context가 700g까지 무감점 허가처럼 작동하면 실패다.
```

현재 closeout 후 guard:

```text
- adaptive target은 같은 kcal 안에서 carb/fat 교환만 반영한다.
- target/TDEE overage는 숨기지 않는다.
- high-volume context는 carb curve context이며 flat no-penalty permission이 아니다.
- trend limiter는 `continuous_recency_weighted_excess` pressure model이다.
- strong/extreme repeated high-carb pressure가 커질수록 adaptive movement가 줄어든다.
- 별도 floor를 두지 않아 극단 반복 이력에서는 movement가 0에 가까워질 수 있다.
```

판단:

```text
adaptive target overfitting risk: release blocker로 남기지 않음
이유: ca09860에서 continuous pressure limiter와 target/TDEE overage guard가 들어감
남은 UX/copy 설명 개선: optional future gate
```
