## 4. blocked record handling

차단 record는 report-only object에 nested preview shell이 붙어도 숫자가 열리지 않는다.

```text

testLocalComponentPoints: null
testLocalBlockedReason: reason
```

이번 suite는 one-meal / insufficient record가 이 형태를 유지하는지 확인한다.
