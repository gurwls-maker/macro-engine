## 1. 구현한 suite

추가 suite:

```text
runMacroRangeExplicitNormalizerSimulationDesignTests
```

등록 위치:

```text
index.html window export
tools/render_audit/run_internal_tests.cjs core profile
package.json test:macro-policy
```

중요:

```text
window export는 suite 함수만 한다.
simulation helper는 suite 내부 local helper로만 존재한다.
normalizeMacroRangeContractForSnapshot production 함수는 만들지 않는다.
```

## 2. test-local helper

suite 내부 helper:

```text
simulateMacroRangeContractNormalization(rawContract, fixedSnapshot, options)
```

이 helper의 역할:

```text
absent / valid / invalid / unsupported 상태를 test-local로 분리
known contract 후보의 allowlist / forbidden field / unknown field pruning 기준 확인
invalid contract가 fixed snapshot을 깨지 않는지 확인
valid simulation output도 저장하지 않는다는 계약 확인
```

이 helper가 하지 않는 일:

```text
normalizeGoalSnapshot 연결
goalSnapshot 저장 field 추가
backup payload 생성
Recent basis 생성
score 계산
UI copy 생성
DailyCoach action 변경
```
