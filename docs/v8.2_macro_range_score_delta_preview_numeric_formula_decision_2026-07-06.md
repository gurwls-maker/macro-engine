## 5. formula definition

후보 formula:

```text
scoreDeltaPreview = candidateScorePreview - fixedScoreSummary.score
```

value domain:

```text
integer -100..100
null
```

예시:

```text
candidateScorePreview 100, fixedScore 81 -> +19
candidateScorePreview 85, fixedScore 81 -> +4
candidateScorePreview 81, fixedScore 81 -> 0
candidateScorePreview 68, fixedScore 81 -> -13
candidateScorePreview 40, fixedScore 81 -> -41
candidateScorePreview 0, fixedScore 100 -> -100
candidateScorePreview 100, fixedScore 0 -> +100
```

중요:

```text
positive delta는 positive bonus가 아니다.
negative delta는 production penalty가 아니다.
zero delta는 blocked sentinel 0이 아니다.
```

## 6. sign convention

sign 의미:

```text
positive: candidate preview score is higher than fixed production score
negative: candidate preview score is lower than fixed production score
zero: candidate preview score equals fixed production score
```

해석 경계:

```text
positive != user-facing improvement copy approved
negative != user-facing regression copy approved
zero != no-action DailyCoach policy approved
```

결정:

```text
sign convention은 numeric comparison meaning만 정한다.
UI label, color, copy, badge, DailyCoach action은 별도 visibility / action gate에서만 정한다.
```

## 7. rounding decision

rounding policy:

```text
identity / no-op integer
```

이유:

```text
candidateScorePreview sum v0는 integer 0..100이다.
fixedScoreSummary.score는 integer 0..100이어야 한다.
integer - integer 결과는 이미 integer다.
따라서 정상 입력에서는 Math.round / floor / ceil이 필요 없다.
```

fractional input 처리:

```text
fractional candidateScorePreview는 invalid다.
fractional fixedScoreSummary.score는 invalid다.
fractional scoreDeltaPreview를 rounding해서 저장하거나 표시하지 않는다.
```

결정:

```text
rounding은 invalid fractional input을 허용하기 위한 보정 장치가 아니다.
fractional input은 scoreDeltaPreview null / violation으로 막는다.
```

## 8. clamp decision

delta formula에는 별도 clamp를 두지 않는다.

이유:

```text
candidateScorePreview가 integer 0..100으로 검증된다.
fixedScoreSummary.score가 integer 0..100으로 검증된다.
따라서 delta는 자연스럽게 -100..100 범위에 들어간다.
```

금지:

```text
candidateScorePreview > 100을 delta clamp로 흡수
fixedScoreSummary.score < 0을 delta clamp로 흡수
out-of-range input을 -100..100으로 조용히 보정
```

결정:

```text
out-of-range input은 scoreDeltaPreview null / violation이다.
delta clamp는 invalid source를 복구하기 위한 장치가 아니다.
```
