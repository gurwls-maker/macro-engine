## 2. formula identity


정책:


## 3. value domain decision

pointsPreview 숫자 후보의 domain:

```text
integer
<= 0
```

허용:

```text
0: eligible neutral / explanation component contribution
negative integer: eligible penalty component contribution
null: blocked / unresolved / non-score eligibility state
```

금지:

```text
positive bonus
fractional points
blocked state sentinel 0
dataQuality sentinel 0
fixed score component clone
previous pointsPreview reuse
null을 UI에서 0점으로 표시
null을 Recent aggregation 0으로 집계
null을 DailyCoach no-action severity로 해석
```

중요:

```text
0은 eligible component의 computed neutral contribution일 수 있다.
0은 blocked / unresolved / dataQuality sentinel이 아니다.
```

## 4. eligibility precondition

pointsPreview 숫자 후보가 될 수 있는 precondition:

```text
dataQuality.status === full_day_candidate
oldGoalSnapshot 또는 fixed target basis 충분
detailed macro data sufficient
rangePolicyCandidate valid
component status resolved
component lane known
storage / UI / Recent / DailyCoach no-leak guard 유지
```

숫자 후보가 될 수 없는 경우:

```text
one_meal_pending
insufficient_record
snapshotless_debug_only
blocked_by_impossible_state
simple record
weight_only record
unsupported range basis
invalid range basis
range policy missing
component status unresolved
```

blocked / unresolved output:

```text
pointsPreview: null
pointsPreviewStatus: "blocked" | "not_computed_policy_unresolved"
```

## 5. lane weight principle

이번 formula v0의 lane 원칙:

```text
anchor lane > range_evidence lane
strong penalty > material penalty > observation penalty
neutral / explanation = 0
dataQuality = eligibility gate, not score component
```

component lane:

```text
kcal: anchor
protein: anchor
fat: range_evidence
carbExchange: range_evidence
dataQuality: eligibility
```

severity lane:

```text
strong: -12 to -24 range
material: -6 to -16 range
observation: -5
neutral / explanation: 0
blocked / unresolved: null
```

주의:

```text
lane weight는 component contribution 후보일 뿐 production score formula가 아니다.
candidateScorePreview 합산은 별도 gate다.
```

## 12. candidateScorePreview relationship

