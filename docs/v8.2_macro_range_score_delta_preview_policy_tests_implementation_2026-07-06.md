## 4. candidate-null guard

eligible fixture:

```text
baseline fixed score only
eligible anchor penalty
eligible range evidence
range policy missing
```

확인:

```text
fixedScoreSummary.score는 comparison input으로 유지
comparison.fixedScore는 fixed score만 참조
rangeScoreCandidateSummary.candidateScorePreview null
comparison.candidateScorePreview null
comparison.scoreDeltaPreview null
candidate_score_numeric_policy decision 유지
```

즉 fixed score가 있어도 candidate score fallback이나 score delta 계산을 하지 않는다.

## 5. unapproved numeric payload guard

suite 내부에서 synthetic unapproved payload를 만든다.

포함:

```text
rangeScoreCandidateSummary.candidateScorePreview: number
comparison.candidateScorePreview: number
componentPreviews.kcal.pointsPreview: number
comparison.scoreDeltaPreview: null
```

확인:

```text
candidateScorePreview 숫자가 payload에 있어도 scoreDeltaPreview 계산 없음
component pointsPreview 숫자가 payload에 있어도 scoreDeltaPreview 계산 없음
unapproved numeric payload는 guard violation으로 잡힘
delta는 계속 null
```

이 fixture는 future production output이 아니라 promoted / unapproved payload를 잡기 위한 negative fixture다.
