## 1. 현재 허용된 위치

현재 숫자 후보가 존재할 수 있는 위치:

```text
buildMacroRangeScoreCandidateReportOnly()
  -> testLocalNumericPreview.testLocalCandidateScorePreview
  -> testLocalNumericPreview.testLocalComponentPoints
```

허용 성격:

```text
testLocalOnly: true
persisted: false
uiVisible: false
productionScoreChanged: false
scoreVersionChanged: false
snapshotSchemaChanged: false
```

이 값은 saved record 점수, Records 표시, backup payload, Recent aggregation, DailyCoach action, Today UI score로 승격되지 않는다.

## 3. consumer map

### 3.1 getDailyAdherenceScore

현재 production score source of truth는 fixed target 기반 `getDailyAdherenceScore()`다.

consumer rule:

```text
getDailyAdherenceScore는 buildMacroRangeScoreCandidateReportOnly output을 읽지 않는다.
testLocalNumericPreview를 percent / detail.componentScores / reasons / label에 반영하지 않는다.
ADHERENCE_SCORING_VERSION은 v6.1_alcohol_penalty_v2로 유지한다.
```

금지:

```text
testLocalCandidateScorePreview -> percent
testLocalComponentPoints -> detail.componentScores
testLocalReasons -> reasons
testLocalCandidateScorePreviewVersion -> scoringVersion
```

### 3.2 backup / restore

현재 full backup은 settings, sorted records, inbody records, presets, meal templates, today calculation drafts, weekly training state를 직렬화한다.

consumer rule:

```text
buildFullBackupPayload는 report-only candidate output을 payload source로 사용하지 않는다.
normalizeFullBackupPayload / normalizeRecordsCollection / normalizeRecord는 testLocalNumericPreview를 persisted field로 복원하지 않는다.
```

금지:

```text
records[].testLocalNumericPreview
records[].testLocalCandidateScorePreview
records[].goalSnapshot.testLocalNumericPreview
backup.data.testLocalNumericPreview
backup.data.records[].candidateScorePreview
```

### 3.3 normalizeGoalSnapshot / normalizeRecord

현재 `normalizeGoalSnapshot()`은 fixed snapshot allowlist만 반환한다.

consumer rule:

```text
goalSnapshot은 targetCal / protein / carbs / fat fixed target 계약을 유지한다.
normalizeRecord는 goalSnapshot normalize 결과와 fixed adherence field만 보존한다.
testLocalNumericPreview는 record나 snapshot schema에 추가하지 않는다.
```

금지:

```text
goalSnapshot.testLocalNumericPreview
goalSnapshot.candidateScorePreview
goalSnapshot.scoreDeltaPreview
record.testLocalNumericPreview
record.rangeScoreCandidate
record.macroRangeContract
```

### 3.4 Recent

현재 Recent component counts는 normalized record summary의 fixed target과 consumed macros를 기반으로 한다.

consumer rule:

```text
getRecentContextRecordSummary는 record meals와 fixed goalSnapshot만 읽는다.
getRecentContextComponentCounts는 targetProtein / targetCarbs / targetFat fixed fields만 집계한다.
testLocalNumericPreview를 Recent component count나 basis summary로 사용하지 않는다.
```

금지:

```text
testLocalComponentPoints -> proteinLowDays / fatLowDays / carbLowTrainingDays
testLocalCandidateScorePreview -> Recent score trend
testLocalCandidateScoreBasisLabel -> Recent basis label
```

### 3.5 DailyCoach

현재 DailyCoach는 signal list와 recent context guard를 기반으로 copy/action을 만든다.

consumer rule:

```text
generateDailyCoachOutput는 testLocalNumericPreview를 action source로 사용하지 않는다.
getDailyCoachSignals는 fixed score / fixed macro balance / Recent context 기준을 유지한다.
```

금지:

```text
testLocalReasons -> coach reason
testLocalComponentPoints -> coach severity
testLocalCandidateScorePreview -> coach confidence
testLocalBlockedReason -> coach action
```

### 3.6 UI / rendered body text

현재 `testLocalNumericPreview.uiVisible`은 false다.

consumer rule:

```text
Today / Records / Recent UI는 testLocalNumericPreview 값을 표시하지 않는다.
debug text나 hidden rendered body text에도 testLocalCandidateScorePreview를 흘리지 않는다.
```

금지:

```text
testLocalCandidateScorePreview display
testLocalComponentPoints display
testLocalCandidateScoreBasisLabel display
range score preview badge
score delta preview text
```

### 3.7 Records state

현재 report-only candidate는 Records state mutation 대상이 아니다.

consumer rule:

```text
buildMacroRangeScoreCandidateReportOnly 실행은 recordsState.items를 바꾸지 않는다.
saved record에는 testLocalNumericPreview가 저장되지 않는다.
old fixed Records는 range preview basis로 조용히 재해석되지 않는다.
```

금지:

```text
recordsState.items[].testLocalNumericPreview
recordsState.items[].candidateScorePreview
recordsState.items[].adherencePercent overwrite
recordsState.items[].adherenceScoringVersion overwrite
```
