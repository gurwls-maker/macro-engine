## 2. field name decision

예약 이름:

```text
candidateScorePreview
```

의미:

```text
range score candidate의 transient numeric preview
fixed production score를 대체하지 않는 비교 후보
저장 점수가 아닌 계산 후보
```

이름을 유지하는 이유:

```text
candidate = production score가 아님
Preview = 저장 계약이 아님
Score = user-facing 점수로 오해될 수 있으므로 위치와 nullability를 엄격히 제한해야 함
```

금지 이름:

```text
score
finalScore
newScore
rangeScore
adherenceScore
savedScore
productionScore
```


## 4. nullability decision

기본값:

```text
candidateScorePreview: null
```

null을 유지해야 하는 경우:

```text
numeric policy unresolved
dataQuality blocked
one_meal_pending
macro_data_insufficient
snapshotless record
simple record
weight_only record
range policy missing
unsupported range basis
invalid range basis
old fixed Records display/storage policy unresolved
UI visibility policy unresolved
scoreDelta policy unresolved
```

future 숫자 후보가 될 수 있는 경우:

```text
full_day_candidate
fixed goalSnapshot present
detailed macro data sufficient
range policy candidate valid
kcal / protein anchor data available
fat / carbExchange evidence available
score/storage separation accepted
no-persist / no-overwrite guard present
```

이 조건은 production 구현 승인이 아니다. 숫자화 전에 test-only guard가 필요하다.

## 5. blocked behavior decision

blocked state에서는:

```text
candidateScorePreview: null
candidateScorePreviewStatus: blocked | not_computed_policy_unresolved
```

blocked reason은 numeric field가 아니라 reason/status field로 표현한다.

후보 reason:

```text
one_meal_pending
macro_data_insufficient
snapshotless_record
simple_record_not_numeric_candidate
weight_only_record_not_numeric_candidate
unsupported_range_basis
invalid_range_basis
numeric_policy_unresolved
ui_visibility_unresolved
score_persistence_unapproved
```

금지:

```text
blocked record에 0점 저장
blocked record에 이전 candidateScorePreview 재사용
blocked record에 fixed score를 candidateScorePreview로 복사
blocked reason을 DailyCoach action으로 연결
```

## 6. old fixed Records behavior

old fixed Records는:

```text
stored adherencePercent 유지

candidateScorePreview 저장 없음
scoreDeltaPreview 저장 없음
pointsPreview 저장 없음
```

허용:

```text
sufficient detailed old fixed Record를 transient candidateScorePreview eligibility 후보로 분류
test-only fixture에서 preview 가능성 관찰

```

금지:

```text
old fixed Records saved adherencePercent overwrite
old fixed Records adherenceScoringVersion overwrite

old fixed Records UI에 candidateScorePreview를 saved score처럼 표시

```
