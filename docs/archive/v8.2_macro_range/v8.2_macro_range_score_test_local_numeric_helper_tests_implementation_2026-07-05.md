## 3. numeric policy shape

이번 helper의 숫자는 production formula가 아니다.

성격:

```text
deterministic test-local preview
penalty-only
0..100 clamp
no positive component points
no bonus from inside_recommended
no bonus from exchange_explained
kcal mismatch cannot be offset by fat/carb range
protein low cannot be offset by fat/carb exchange
dataQuality is eligibility gate
```

version / label:

```text
testLocalCandidateScorePreviewVersion = range_score_candidate_preview_test_local_v0
testLocalCandidateScoreBasisLabel = macro_range_candidate_preview_test_local
testLocalFixedScoreBasisLabel = fixed_adherence:v6.1_alcohol_penalty_v2
```

## 4. eligibility

숫자 허용:

```text
full_day_candidate fixture
sufficient detailed old fixed Record
fixed goalSnapshot present
macro data sufficient
```

숫자 차단:

```text
simple record
weight_only record
snapshotless record
one_meal_pending
macro_data_insufficient
```

차단된 record는 아래처럼 남는다.

```text
testLocalCandidateScorePreview: null
testLocalComponentPoints: null
testLocalBlockedReason: reason
```

## 5. no-leak guard

```text

backup payload에 candidateScorePreview가 들어가지 않음
Recent output unchanged
DailyCoach output unchanged
UI text unchanged
Records state unchanged
ADHERENCE_SCORING_VERSION unchanged
```
