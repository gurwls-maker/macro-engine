## 2. test-local helper

suite 내부 helper:

```text
normalizeMacroRangeContractForDesignTest(rawContract, fixedSnapshot, options)
```

역할:

```text
absent / valid / invalid / unsupported contract 후보 분류
known allowlist field만 normalizedContract 후보로 반환
fixed snapshot survival rule 검증
forbidden score/debug/copy/evidence field 차단
unknown field policy를 prune_with_warning 또는 reject로 명시
```

하지 않는 일:

```text
production normalizeGoalSnapshot 연결
goalSnapshot 저장 field 추가
backup payload 생성
Recent basis 생성
score 계산
UI copy 생성
DailyCoach action 생성
```

## 3. 추가 테스트

`runMacroRangeContractNormalizerDesignTests`는 12개 케이스로 구성했다.

```text
1. absent contract leaves fixed snapshot valid
2. valid known contract returns normalizedContract candidate without persistence
3. valid contract does not mutate fixed target fields
4. targetCal mismatch returns invalid and fixed snapshot survives
5. protein order violation returns invalid and fixed snapshot survives
6. fat order violation returns invalid and fixed snapshot survives
7. carb order violation returns invalid and fixed snapshot survives
8. unsupported version returns unsupported and fixed snapshot survives
9. unknown field policy is explicit: default warn/prune, reject option invalidates
10. forbidden candidateScorePreview invalidates contract and never persists
11. appliesToSavedRecord true invalidates contract
12. helper output has zero production, storage, backup, Recent, score, and UI connections
```
