## 3. full-day candidate guard

`full_day_candidate`는 숫자 preview 후보로 식별될 수는 있지만, 아직 숫자로 계산하지 않는다.

유지되는 값:

```text
candidateScorePreview: null
scoreDeltaPreview: null
component pointsPreview: null
candidateScorePreviewStatus: not_computed_policy_unresolved
```

즉 full-day 조건은 numeric score readiness가 아니라, 이후 policy decision이 닫힌 뒤에만 검토할 수 있는 후보 상태다.

## 4. scoreDeltaPreview gate

`scoreDeltaPreview`는 이번에도 마지막 gate로 남긴다.

이유:

```text
fixed score와 candidate score를 직접 비교한다.
production score 개선 / 악화처럼 읽힐 수 있다.
old/new Records basis 비교 의미를 만든다.
UI / DailyCoach / Recent로 번지기 쉽다.
```

따라서 `candidateScorePreview`가 null인 동안 `scoreDeltaPreview`도 반드시 null이다.
