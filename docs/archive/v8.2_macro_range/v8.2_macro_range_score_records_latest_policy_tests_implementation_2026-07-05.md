suite 내부에만 다음 분류 helper를 둔다.

```text

```

이 helper는 test-only guard다.

```text
production helper 아님
numeric score helper 아님
storage helper 아님
UI helper 아님

```

helper output에는 production-facing score field를 넣지 않는다.

```text

```

## 4. eligibility matrix

허용 후보:

```text
detailed old fixed Record
fixed goalSnapshot present
mealCount >= 2
macro data sufficient
latest-policy preview only
non-persisted
```

차단 또는 unavailable:

```text
simple record
weight_only record
snapshotless record
one_meal_pending
macro_data_insufficient
raw import / raw backup 상태 그대로의 record
```

## 5. no-mutation guard

suite는 raw record와 goalSnapshot에 forbidden candidate fields를 일부러 넣는다.

```text
macroRangeContract
macroRangeBasisSignature
rangeScoreCandidate
candidateScorePreview
pointsPreview
scoreDeltaPreview
candidateScoreBasisLabel
fixedScoreBasisLabel
```

기대:

```text
normalizeRecord() 후 남지 않음
normalizeGoalSnapshot() 후 남지 않음
buildFullBackupPayload() 후 남지 않음
saved adherencePercent 유지
saved adherenceScoringVersion 유지
```
