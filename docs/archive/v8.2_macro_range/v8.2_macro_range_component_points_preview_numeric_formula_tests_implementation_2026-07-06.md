## 4. exact value guard

suite 내부 formula candidate helper가 확인하는 값:

```text
kcal matched 0
kcal high / low -10
kcal severe high / severe low -16
protein satisfied 0
protein low -16
protein severe low -24
fat inside_recommended 0
fat below_recommended / above_recommended -5
fat below_action_floor / above_flexible -12
carbExchange exchange_explained / inside_candidate_range 0
carbExchange below_training_support -8
carbExchange above_candidate_range -6
dataQuality any status null
unresolved / blocked null
```

검증 기준:

```text
component + status + evidence별 exact expected value를 source of truth로 둔다.
severity label만으로 전역 정렬하지 않는다.
anchor material 값이 range evidence strong 값보다 클 수 있음을 허용한다.
```
