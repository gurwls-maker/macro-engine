## 4. exact formula guard

suite 내부 formula candidate helper가 확인하는 값:

```text
[0, 0, 0, 0] -> raw 100 -> candidate 100
[-10, 0, -5, 0] -> raw 85 -> candidate 85
[-16, -24, -12, -8] -> raw 40 -> candidate 40
[-10, -16, 0, -6] -> raw 68 -> candidate 68
[-16, 0, -5, -6] -> raw 73 -> candidate 73
```

확인:

```text
candidateScorePreview 후보는 integer 0..100
dataQuality는 합산 제외
severity label이 아니라 component point exact value를 합산
```

## 5. clamp / rounding guard

lower clamp fixture:

```text
[-80, -40, -30, -10] -> raw -60 -> candidate 0
[-120, 0, 0, 0] -> raw -20 -> candidate 0
```

계속 금지:

```text
blocked 상태를 0으로 표현
positive component point를 100으로 조용히 upper clamp
fractional component point를 round / floor / ceil로 복구
NaN / Infinity / string numeric value 허용
```
